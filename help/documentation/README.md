# Kokobrain — User Guide

Welcome to **Kokobrain**, a desktop note-taking app inspired by [Obsidian.md](https://obsidian.md). Kokobrain stores your notes as plain Markdown files (`.md`) in a folder on your computer — no cloud, no lock-in. You own your data.

This guide will walk you through every feature, from creating your first note to building a connected knowledge base.

## Where to Start

If you're new to Kokobrain, follow this path:

1. **[Getting Started](01-getting-started.md)** — Open a vault and create your first note
2. **[The Editor](03-editor.md)** — Understand tabs, modes, and the status bar
3. **[Markdown Guide](04-markdown.md)** — Learn to format your notes
4. **[Wikilinks](05-wikilinks.md)** — Connect your notes together
5. **[Search & Navigation](06-search-and-navigation.md)** — Find anything fast

After that, explore any topic that interests you.

## Conventions Used in This Guide

- **`Cmd+P`** — Press these keys simultaneously (macOS Command key + P).
- **`[[note name]]`** — Text you type literally in the editor.
- Code blocks show exact syntax to use.
- Callout boxes highlight important information:

> [!TIP]
> Power-user tricks and shortcuts.

> [!NOTE]
> Important context or clarification.

> [!WARNING]
> Actions that are hard to undo or may cause data loss.
