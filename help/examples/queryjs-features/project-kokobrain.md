# Projeto Kokobrain

A desktop note-taking app inspired by Obsidian em Svelte 5 + Tauri 2.

## Stack
- Frontend: Svelte 5, SvelteKit, TypeScript
- Backend: Tauri 2 (Rust)
- UI: shadcn-svelte
